# Figma MCP

**Назначение:** Работа с дизайнами в Figma — чтение, экспорт, комментарии.

**Тип:** HTTP (облачный)
**URL:** https://mcp.figma.com/mcp
**Автор:** Figma официальный ✅

## Установка

```bash
claude mcp add --transport http --scope user figma https://mcp.figma.com/mcp
```

## Использование

Передать дизайн из Figma агенту для реализации в коде.

## Статус безопасности

✅ БЕЗОПАСНО — официальный облачный сервис
