# Notion MCP

**Назначение:** Создание, чтение, поиск страниц в Notion.

**Тип:** HTTP (облачный)
**URL:** https://mcp.notion.com/mcp
**Автор:** Notion официальный ✅

## Установка

```bash
claude mcp add --transport http --scope user notion https://mcp.notion.com/mcp
```

## Использование

Сохранение заметок, управление задачами, базы знаний.
